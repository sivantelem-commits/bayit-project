# 🏠 ניהול בניית הבית

מערכת ניהול מקיפה לפרויקט בניית הבית — מבחירת אנשי מקצוע ועד כניסה לבית.

## ✨ פיצ׳רים
- דאשבורד — סקירה כללית, ניצול תקציב, גרפים
- תקציב והוצאות — מעקב חשבוניות לפי קטגוריות
- לוח זמנים — גאנט, שלבי בנייה, מעקב התקדמות
- אנשי מקצוע — ניהול קבלנים, תשלומים, דירוגים
- מסמכים והיתרים — ארכיון עם התראות תפוגה
- תמונות — גלריה מאורגנת לפי מיקום ושלב
- יומן עבודה — לוג יומי מה נעשה באתר
